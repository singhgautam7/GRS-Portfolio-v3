A modern, privacy-first expense manager built to be fast, offline-first and
actually enjoyable to use. Smart transaction entry, category budgets with
real-time progress, recurring transaction automation, powerful history search
and filtering, loan/investment tracking, and CSV/PDF export. All data is stored
locally on-device using Isar, with no accounts or cloud dependency.
